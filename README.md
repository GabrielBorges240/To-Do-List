# To-Do List com Prioridades

###   Demo
 https://gabrielborges240.github.io/To-Do-List/

---

## Descrição

Aplicação web moderna e responsiva para gerenciamento de tarefas, com suporte a **níveis de prioridade**, **busca por nome e data**, **modo claro/escuro**, **salvamento automático** e **exportação em PDF**.

---

## Funcionalidades

- Adicionar tarefas com nome, data (ano limitado a 4 dígitos) e nível de prioridade (Baixa, Média ou Alta)  
- Marcar tarefas como concluídas  
- Excluir tarefas individualmente ou limpar todas de uma vez  
- Busca inteligente por nome ou data  
- Barra de progresso com porcentagem concluída  
- Alternância entre modo claro e escuro (com salvamento automático)  
- Exportar todas as tarefas para PDF  
- Interface responsiva e moderna  
- Salvamento automático das tarefas no navegador (LocalStorage)

---

## Tecnologias utilizadas

- **HTML5**  
- **CSS3** (com gradientes animados e design responsivo)  
- **JavaScript puro (Vanilla JS)**  
- **jsPDF** – geração de relatórios em PDF  
- **LocalStorage** – persistência de dados localmente
