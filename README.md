# To-Do List

Lista interativa de tarefas para organização diária.

Demo:    https://gabrielborges240.github.io/To-Do-List/

## Funcionalidades
- Adicionar e remover tarefas
- Marcar como concluída
- Armazenamento local

## Tecnologias
- HTML5
- CSS3
- JavaScript
- LocalStorage

## Como executar
Abra o arquivo `index.html` no navegador.
